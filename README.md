# Healthcare Patient, Treatment & Analytics System (MySQL)

This project is designed to manage and analyze healthcare data using MySQL.
It focuses on patient records, doctor information, appointments, and treatments
to generate meaningful healthcare analytics and reports.

The system helps analyze treatment costs, doctor workload, and patient history,
which are critical for hospital management and healthcare decision-making.

## Technologies Used
- MySQL 8.0
- SQL

## SQL Concepts Covered
- Relational Database Design
- Primary & Foreign Keys
- Joins
- Aggregate Functions
- Group By
- Date Functions

## Key Features
- Patient and doctor data management
- Appointment tracking
- Treatment cost analysis
- Doctor-wise patient load analysis

## Use Case
Useful for hospitals, clinics, and healthcare analytics systems to monitor
treatment data and improve operational efficiency.
